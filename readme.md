# FST UPITRA — Landing Page

Halaman profil Fakultas Sains dan Teknologi Universitas Pignatelli Triputra.

## Fitur

- Info fakultas dan program studi (S1 Sistem Informasi, Informatika, Software Engineering)
- Kegiatan mahasiswa per prodi — data diambil langsung dari API UPITRA
- Modal preview foto kegiatan
- Responsive & sticky navbar

## Cara Pakai

Buka `index.html` langsung di browser, tidak perlu build tool atau dependencies.

## Struktur

```
index.html   → semua HTML, CSS, dan JS dalam satu file
```

## API

Data kegiatan diambil dari:
```
https://new.upitra.ac.id/api/prodi/
```
